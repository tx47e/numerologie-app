# Numerologie App

Proiect pentru construirea unei aplicatii de numerologie in limba romana.

Scopul initial este crearea unei baze de informatii bine structurate, care poate
fi folosita ulterior de aplicatie pentru afisare, cautare, interpretari si
generare de rapoarte.

## Structura curenta

- `vibratii/`: baza de continut pentru vibratiile numerologice 1-9.
- `docs/`: documentatie de proiect, decizii si pasi de lucru.

## Directie de lucru

1. Construim baza de continut pentru vibratiile 1-9.
2. Stabilim modelul de date necesar aplicatiei.
3. Alegem structura aplicatiei si tehnologia de implementare.
4. Transformam continutul Markdown intr-o sursa usor de consumat de aplicatie.
5. Construim interfata si fluxurile de calcul numerologic.
