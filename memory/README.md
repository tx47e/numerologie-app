# Memory

Acest director va contine note persistente despre decizii, preferinte si context
de lucru.

Scopul este sa pastram lucrurile care trebuie amintite intre etape:

- decizii editoriale;
- decizii tehnice;
- preferinte de structura;
- reguli de lucru;
- observatii importante pentru dezvoltare.

## Preferinte editoriale

- La fiecare interpretare, tonul trebuie sa fie clar, academic si usor de
  digerat de catre o persoana care nu are cunostinte despre numerologie.
- Explicatiile trebuie sa ramana accesibile: conceptele numerologice se introduc
  gradual, fara jargon neexplicat si fara formulari rigide sau profetice.
- Toate explicatiile trebuie sa functioneze ca un proces de coaching: sa arate
  clar partile pozitive si partile negative ale unei vibratii, dar sa orienteze
  cititorul spre constientizare, integrare si evolutie superioara.
- Partea de coaching trebuie formulata astfel incat persoanei pentru care se face
  lucrarea sa i se puna la dispozitie o decizie constienta si sfaturi practice.
  Interpretarea trebuie sa ajute omul sa vada optiunile, consecintele posibile si
  pasii concreti prin care poate lucra cu energia descrisa.
