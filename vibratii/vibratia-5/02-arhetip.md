# Vibratia 5 - Arhetip

Arhetipul vibratiei 5 este Exploratorul. Exploratorul traverseaza granite, testeaza
posibilitati si aduce informatii din teritorii noi.

In forma matura, Exploratorul este flexibil, viu si inteligent adaptativ. In forma
imatura, poate deveni instabil, nelinistit si dependent de stimulare.
