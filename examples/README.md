# Examples

Acest director va contine exemple de lucru pentru aplicatie.

Lucrarile pentru persoane se organizeaza in directoare dedicate, denumite dupa
data nasterii si numele persoanei, in format:

```text
AAAA-LL-ZZ-nume-prenume
```

In fiecare director se pastreaza lucrarea curenta pentru persoana respectiva si,
daca este cazul, fisiere auxiliare.

Exemple posibile:

- profiluri numerologice complete;
- template de lucrare numerologica completa;
- date de nastere folosite pentru test;
- exemple de nume;
- exemple de interpretari generate;
- exemple de rapoarte finale.
