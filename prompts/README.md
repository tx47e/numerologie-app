# Prompts

Acest director va contine prompturi folosite pentru generare, analiza, reformulare
sau asistenta in aplicatie.

Exemple posibile:

- prompt pentru generarea unei interpretari numerologice;
- prompt pentru sumarizarea unui profil;
- prompt pentru adaptarea tonului;
- prompt pentru explicatii scurte versus explicatii ample.
