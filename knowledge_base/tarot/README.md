# Tarot

Acest director contine baza simbolica pentru tarot, separata de calculele
numerologice. Scopul este sa evitam repetarea acelorasi interpretari in fiecare
calcul si sa pastram tarotul ca sursa comuna de analogii.

## Structura

- `arcane-majore/`: interpretari pentru Arcanele Majore folosite in analogiile numerologice.
- `arcane-minore/`: structura pentru Arcanele Minore, organizate pe suite si elemente.

## Suite si elemente

- Bate: foc, vointa, actiune, initiativa, energie vitala.
- Cupe: apa, emotii, relatie, receptivitate, memorie afectiva.
- Sabii: aer, gandire, decizie, conflict, claritate mentala.
- Monede: pamant, corp, resurse, munca, stabilitate materiala.

## Regula de lucru

Tarotul nu inlocuieste calculul numerologic. El adauga un strat arhetipal:
imagine, rol, plusuri, minusuri si chei de integrare.
