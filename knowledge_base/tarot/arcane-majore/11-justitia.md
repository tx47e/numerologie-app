# 11 - Justice / Justitia

Justitia arata echilibrul, consecintele, adevarul obiectiv si responsabilitatea
alegerilor. Este arcana cantaririi lucide: ce este corect, ce este datorat si ce
trebuie aliniat.

## Plusuri

- claritate;
- corectitudine;
- discernamant;
- responsabilitate;
- echilibru;
- respect pentru adevar.

## Minusuri

- judecata rece;
- rigiditate;
- criticism;
- pedepsire;
- lipsa de compasiune;
- blocaj in analiza.

## Cheie de integrare

Justitia cere adevar fara cruzime si responsabilitate fara impietrire.
