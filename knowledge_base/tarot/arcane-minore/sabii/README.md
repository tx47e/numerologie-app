# Sabii - Aer

Sabiile corespund elementului aer: gandire, cuvant, claritate, decizie, conflict,
analiza si adevar mental.

## Directii simbolice

- discernamant;
- comunicare;
- strategie;
- taiere;
- alegere;
- confruntare cu adevarul.

## Umbra

- anxietate;
- criticism;
- conflict;
- raceala;
- supragandire;
- cuvinte care ranesc.
