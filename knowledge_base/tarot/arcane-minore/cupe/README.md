# Cupe - Apa

Cupele corespund elementului apa: emotii, relatii, receptivitate, memorie
afectiva, iubire, vindecare si imaginatie.

## Directii simbolice

- atasament;
- sensibilitate;
- intimitate;
- dorinta de apropiere;
- inspiratie emotionala;
- procese de vindecare.

## Umbra

- dependenta emotionala;
- idealizare;
- confuzie afectiva;
- victimizare;
- nostalgie excesiva;
- lipsa de limite.
