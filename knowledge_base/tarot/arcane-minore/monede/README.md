# Monede - Pamant

Monedele corespund elementului pamant: corp, resurse, bani, munca, stabilitate,
ritm, valoare si materializare.

## Directii simbolice

- constructie;
- rabdare;
- siguranta;
- competenta;
- rezultate concrete;
- grija fata de corp si resurse.

## Umbra

- stagnare;
- atasament material;
- frica de lipsa;
- rigiditate;
- posesivitate;
- reducerea valorii la bani sau statut.
