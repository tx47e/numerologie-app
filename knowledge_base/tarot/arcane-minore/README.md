# Arcane Minore

Arcanele Minore descriu experienta concreta: actiuni, emotii, ganduri, resurse,
conflicte, relatii si situatii cotidiene. Ele sunt organizate pe patru suite,
fiecare asociata cu un element.

## Suite

- [Bate - Foc](bate/README.md)
- [Cupe - Apa](cupe/README.md)
- [Sabii - Aer](sabii/README.md)
- [Monede - Pamant](monede/README.md)

## Structura fiecarei suite

Fiecare suita contine:

- cartile numerale: As, 2, 3, 4, 5, 6, 7, 8, 9, 10;
- cartile de curte: Paj, Cavaler, Regina, Rege;
- interpretari pentru plusuri, minusuri si chei de integrare.
