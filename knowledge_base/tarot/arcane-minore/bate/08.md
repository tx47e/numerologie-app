# 8 de Bate

8 de Bate arata viteza, comunicarea rapida, evenimentele care se pun in miscare
si accelerarea unei situatii.

## Plusuri

- progres rapid;
- miscare;
- mesaje;
- aliniere;
- energie directionata.

## Minusuri

- graba;
- supraincarcare;
- reactii impulsive;
- lipsa de procesare.

## Cheie de integrare

Viteza este utila cand directia este clara; altfel devine zgomot si imprastiere.
