# Bate - Foc

Batele corespund elementului foc: vointa, actiune, initiativa, curaj, impuls,
creativitate si energie vitala.

## Directii simbolice

- pornire;
- dorinta;
- miscare;
- risc;
- expresie instinctiva;
- proiecte si actiuni.

## Umbra

- graba;
- conflict;
- impulsivitate;
- ardere rapida;
- lipsa de continuitate;
- dominare prin vointa.
