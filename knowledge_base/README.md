# Knowledge Base

Acest director contine baza de cunoastere numerologica a proiectului.

Tot ce tine de metoda, interpretari, formule, exemple de calcul si continut
numerologic stabil trebuie pastrat aici.

## Structura

- `vibratii/`: interpretari pentru vibratiile de la 1 la 9.
- `calcule/`: metode de calcul, semnificatii si exemple.
- `tarot/`: interpretari simbolice pentru Arcanele Majore si analogii numerologice.
- `exprimari/`: formulari generate si validate pentru interpretari recurente,
  organizate pe teme precum destin, vibratii, punti, scara bunastarii si nume.
- `calcule/relatii/`: metode pentru compatibilitate si omuletul relatiilor.

## Regula de lucru

Continutul din `knowledge_base/` este sursa principala pentru aplicatie. Codul,
prompturile si fisierele generate pot consuma acest continut, dar nu trebuie sa
devina sursa principala a metodei numerologice.

Formularile bune aparute in lucrari se salveaza in `exprimari/`, in fisierul
temei potrivite. Cand exista mai multe variante pentru acelasi rezultat numeric,
se pastreaza variantele utile cu contextul lor, pentru a construi in timp o baza
de exprimari reutilizabile.
