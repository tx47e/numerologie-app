# Vibratia destinului - metoda

Formula adoptata pentru proiect:

```text
vibratia destinului = reducere_numerologica(ziua nasterii + luna nasterii + anul nasterii)
```

## Pasi de calcul

1. Se preia ziua nasterii.
2. Se preia luna nasterii.
3. Se preia anul nasterii.
4. Se aduna cele trei valori ca numere.
5. Rezultatul se reduce la 1-9.

## Observatii

Vibratia destinului este un calcul derivat din data de nastere. In proiect,
trebuie deosebita de `destinul` derivat din nume, care foloseste numarul de
exprimare.
