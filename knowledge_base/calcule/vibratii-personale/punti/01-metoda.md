# Punti - metoda

Puntea este diferenta dintre doua vibratii. Se foloseste valoarea absoluta.

```text
punte = valoare_absoluta(vibratie_a - vibratie_b)
```

## Puntea interior - exterior

```text
punte interior-exterior =
  valoare_absoluta(vibratia_interioara - vibratia_exterioara)
```

Aceasta arata ce ajuta persoana sa faca pace intre cine este in interior si
rolul social pe care il are de jucat.

## Punti catre destin

Se pot calcula punti intre:

- vibratia interioara si vibratia destinului;
- vibratia exterioara si vibratia destinului;
- vibratia cosmica si vibratia destinului;
- alte vibratii importante.

```text
punte interior-destin = |vibratia_interioara - vibratia_destinului|
punte exterior-destin = |vibratia_exterioara - vibratia_destinului|
punte cosmic-destin = |vibratia_cosmica - vibratia_destinului|
```
