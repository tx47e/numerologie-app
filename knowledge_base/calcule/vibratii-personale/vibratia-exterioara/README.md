# Vibratia exterioara

Vibratia exterioara descrie felul in care persoana se exprima in lume: prezenta,
comportament vizibil, imagine si modul in care este perceputa.

In acest proiect, vibratia exterioara este luna nasterii redusa numerologic.

```text
vibratia exterioara = reducere_numerologica(luna nasterii)
```

## Ce arata

- felul in care persoana se manifesta in relatii si contexte sociale;
- imaginea pe care o proiecteaza in exterior;
- modul in care ceilalti pot percepe comportamentul sau;
- tipul de rol social pe care il activeaza natural;
- felul in care calitatile interioare devin vizibile.

## Input necesar

- luna nasterii, intre 1 si 12.

## Rezultat

- o vibratie intre 1 si 9;
- optional, luna 11 poate pastra valoarea intermediara 11 pentru interpretari extinse.
