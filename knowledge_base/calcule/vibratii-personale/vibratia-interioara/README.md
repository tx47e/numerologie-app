# Vibratia interioara

Vibratia interioara descrie dinamica privata a persoanei: motivatie, sensibilitate,
nevoi profunde si felul in care omul se raporteaza la sine.

In acest proiect, vibratia interioara este ziua nasterii redusa numerologic.

```text
vibratia interioara = reducere_numerologica(ziua nasterii)
```

## Ce arata

- felul in care persoana se simte pe sine;
- instinctele si reactiile de baza;
- nevoile emotionale si psihologice;
- dorintele care motiveaza persoana;
- ceea ce o face sa se simta vie, implicata si conectata la propria directie;
- inclinatiile naturale care apar inaintea rolurilor sociale;
- modul in care persoana isi filtreaza experientele din interior.

## Input necesar

- ziua nasterii, intre 1 si 31.

## Rezultat

- o vibratie intre 1 si 9;
- optional, se poate pastra si valoarea intermediara 11 sau 22 pentru interpretari extinse.
