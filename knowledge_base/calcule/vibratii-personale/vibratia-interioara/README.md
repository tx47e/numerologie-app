# Vibratia interioara

Vibratia interioara descrie dinamica privata a persoanei: motivatie, sensibilitate,
nevoi profunde si felul in care omul se raporteaza la sine.

In acest proiect, vibratia interioara este ziua nasterii redusa numerologic.
Ea se citeste impreuna cu [vibratia exterioara](../vibratia-exterioara/README.md),
[vibratia globala](../vibratia-globala/README.md) si
[vibratia destinului](../vibratia-destinului/README.md), deoarece impreuna
descriu modul in care interiorul, aparitia publica si directia de viata se
sprijina sau se tensioneaza reciproc.

```text
vibratia interioara = reducere_numerologica(ziua nasterii)
```

## Ce arata

- felul in care persoana se simte pe sine;
- instinctele si reactiile de baza;
- nevoile emotionale si psihologice;
- dorintele care motiveaza persoana;
- ceea ce o face sa se simta vie, implicata si conectata la propria directie;
- inclinatiile naturale care apar inaintea rolurilor sociale;
- modul in care persoana isi filtreaza experientele din interior.

In lectura practica, vibratia interioara nu se izoleaza de [punti](../punti/README.md),
de [numarul intim](../../nume/numarul-intim/README.md) sau de structurile de
tip [patratul lui Pitagora](../../patratul-lui-pitagora/README.md). Ele adauga
context, dar nu inlocuiesc baza interioara.

## Input necesar

- ziua nasterii, intre 1 si 31.

## Rezultat

- o vibratie intre 1 si 9;
- optional, se poate pastra si valoarea intermediara 11 sau 22 pentru interpretari extinse.

Cand ziua este 10, 20 sau 30, rezultatul final este 1, 2 sau 3, dar traseul
intermediar merita pastrat in observatii. In acel traseu apar praguri precum
10, 11 sau 22, care pot schimba nuanta interpretarii in raport cu
[vibratia exterioara](../vibratia-exterioara/README.md) si cu
[vibratia globala](../vibratia-globala/README.md).
