# Vibratia cosmica - metoda

Vibratia cosmica se calculeaza din anul nasterii. Ea arata menirea spirituala,
valorile, rolurile si locul persoanei in planul mai larg al vietii.

In documentatia bibliografica, vibratia cosmica are trei componente:

- fixa;
- variabila;
- totala.

## Date necesare

- anul nasterii.

## Vibratia cosmica fixa

Vibratia cosmica fixa foloseste primele doua cifre ale anului.

```text
vibratia_cosmica_fixa = primele_doua_cifre_ale_anului
```

Ea este legata de subconstient, emisfera dreapta, planurile subtile si
dimensiunea care ramane constanta.

## Vibratia cosmica variabila

Vibratia cosmica variabila foloseste ultimele doua cifre ale anului, adunate si
reduse numerologic.

```text
vibratia_cosmica_variabila =
  reducere_numerologica(ultimele_doua_cifre_ale_anului)
```

Ea este legata de constient, planurile concrete si ceea ce poate face omul in
aceasta viata.

## Vibratia cosmica totala

Vibratia cosmica totala foloseste toate cifrele anului, adunate si reduse
numerologic.

```text
vibratia_cosmica_totala =
  reducere_numerologica(suma_cifrelor(anul_nasterii))
```

Aceasta este forma folosita anterior in proiect sub denumirea generala de
`vibratia cosmica`.

## Rezultat final

| Componenta | Calcul | Rezultat | Sens |
| --- | --- | --- | --- |
| Fixa | primele doua cifre ale anului |  | plan subtil, constant |
| Variabila | ultimele doua cifre reduse |  | ce poate face omul concret |
| Totala | toate cifrele anului reduse |  | raportare la totalitate |
