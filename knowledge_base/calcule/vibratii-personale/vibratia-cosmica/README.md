# Vibratia cosmica

Vibratia cosmica se calculeaza din anul nasterii. Ea descrie fundalul larg,
contextul generational si energia de timp in care persoana intra in viata.

## Formula

```text
vibratia cosmica = reducere_numerologica(anul nasterii)
```
