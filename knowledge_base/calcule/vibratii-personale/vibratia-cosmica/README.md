# Vibratia cosmica

Vibratia cosmica se calculeaza din anul nasterii. Ea descrie fundalul larg,
contextul generational, menirea spirituala si raportarea persoanei la planul mai
mare al vietii.

## Formula

```text
vibratia cosmica fixa = primele doua cifre ale anului
vibratia cosmica variabila = reducere_numerologica(ultimele doua cifre ale anului)
vibratia cosmica totala = reducere_numerologica(suma cifrelor anului)
```
