# Aspecte de indreptat si solutia

Acest calcul arata aspectele care merita indreptate sau reparate in aceasta
viata si vibratia solutiei de indreptare.

## Calcule premergatoare

Pentru acest calcul sunt necesare:

- [Calea destinului](../calea-destinului/README.md), folosita ca suma a tuturor
  cifrelor din data nasterii, fara reducere la o singura cifra;
- [Vibratia interioara](../vibratia-interioara/README.md), nu ca rezultat
  direct, ci ca zona de lucru in care se verifica ziua nasterii;
- data nasterii completa, pentru extragerea primei cifre din ziua nasterii.

Ordinea recomandata este:

1. Se stabileste data nasterii folosita in lucrare.
2. Se calculeaza [calea destinului](../calea-destinului/README.md).
3. Se extrage prima cifra din ziua nasterii.
4. Se calculeaza aspectele de indreptat.
5. Se reduce rezultatul pentru vibratia solutiei.
