# Aspecte de indreptat - exemple

## Exemplu: 3.2.1980

Calea destinului:

```text
3 + 2 + 1 + 9 + 8 + 0 = 23
```

Prima cifra din ziua de nastere:

```text
3
```

Calcul:

```text
23 - 2 x 3 = 23 - 6 = 17
```

Rezultat:

- aspecte de indreptat: `17`;
- aspectele citite separat: `1` si `7`.

Vibratia solutiei:

```text
1 + 7 = 8
```

Rezultat final:

- vibratia solutiei: `8`.
