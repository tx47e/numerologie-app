# Vibratia globala - metoda

Formula adoptata pentru proiect:

```text
vibratia globala = reducere_numerologica(ziua nasterii + luna nasterii)
```

## Pasi de calcul

1. Se preia ziua nasterii.
2. Se preia luna nasterii.
3. Se aduna ziua si luna ca numere.
4. Rezultatul se reduce la 1-9.

## Observatii

Vibratia globala este sinteza dintre zi si luna. Ea trebuie citita impreuna cu
vibratia interioara si vibratia exterioara, nu in locul lor.
