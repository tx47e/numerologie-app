# Vibratia globala

Vibratia globala se calculeaza din ziua si luna nasterii. Ea sintetizeaza
interiorul si exteriorul persoanei intr-o vibratie comuna.

## Formula

```text
vibratia globala = reducere_numerologica(ziua nasterii + luna nasterii)
```
