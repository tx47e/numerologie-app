# Calea destinului - metoda

Formula adoptata pentru proiect:

```text
calea destinului = suma cifrelor din data nasterii, neredusa complet
```

## Pasi de calcul

1. Se scrie data nasterii in forma `ZZLLAAAA`.
2. Se aduna toate cifrele datei.
3. Rezultatul se pastreaza ca numar compus.

## Observatii

Calea destinului nu se reduce complet la 1-9. Daca este nevoie de vibratia finala
a traseului, se foloseste `vibratia destinului`.
