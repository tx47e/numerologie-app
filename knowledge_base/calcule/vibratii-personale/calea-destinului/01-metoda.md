# Calea destinului - metoda

Formula adoptata pentru proiect:

```text
calea destinului = suma cifrelor din data nasterii, neredusa complet
```

## Pasi de calcul

1. Se scrie data nasterii in forma `ZZLLAAAA`.
2. Se aduna toate cifrele datei.
3. Rezultatul se pastreaza ca numar compus.

## Interpretare

Calea destinului povesteste despre ce si cum are omul de facut prin viata ca sa
isi realizeze destinul. Ea nu arata doar o cifra de rezultat, ci modul in care
persoana poate lucra cu propriul drum.

In bibliografie, calea destinului apare si ca `vibratia caii destinului`.
Termenii sunt tratati ca echivalenti in proiect.

Numarul cel mai mic posibil este `4`, iar cel mai mare este `48`. Aceasta nu este
o scara ierarhica. Numarul arata mai ales raportul dintre perioada de antrenament
pentru viata, sanse, ajutoare si obstacole. Cu cat numarul este mai mic, omul
poate primi mai multe sanse, pentru ca are nevoie de ele ca sa faca ce are de
facut. Cu cat numarul este mai mare, sansele pot fi mai putine sau mai subtile,
pentru ca perioada de antrenament interior este mai mare.

Interpretarea se face prin doua intrebari:

1. Ce are de facut omul?
2. Cum are de facut acel lucru?

Raspunsul la intrebarea `ce?` arata ce calitati trebuie cultivate si ce aspecte
negative trebuie invinse. Raspunsul la intrebarea `cum?` arata felul in care
acele calitati se aplica: prin caracteristicile de polaritate plus ale cifrei,
fara caracteristicile de polaritate minus ale aceleiasi cifre.

Pentru o cale a destinului formata din doua cifre, prima cifra raspunde la
intrebarea `ce?`, iar a doua cifra raspunde la intrebarea `cum?`.

Exemplu:

```text
calea destinului = 34
```

In acest caz:

- `3` raspunde la intrebarea `ce?`;
- `4` raspunde la intrebarea `cum?`.

Persoana are de cultivat calitatile lui `3` si de invins umbrele lui `3`, iar
acest lucru se face prin polaritatea pozitiva a lui `4`, fara rigiditatea,
blocajul sau celelalte caracteristici de polaritate minus ale lui `4`.

Daca calea destinului este formata dintr-o singura cifra, aceeasi cifra raspunde
atat la intrebarea `ce?`, cat si la intrebarea `cum?`.

Exemplu:

```text
calea destinului = 7
```

In acest caz, omul are de cultivat calitatile lui `7`, de invins umbrele lui `7`
si de facut acest lucru tot prin polaritatea pozitiva a lui `7`.

## Categorii de interpretare

Prima cifra a caii destinului arata categoria mare de lucru:

| Interval | Prima cifra | Directie |
| --- | ---: | --- |
| 4-9 | 0 implicit | Prezenta, centrare si atentie la ceea ce vine. |
| 10-19 | 1 | Desavarsire de sine pe toate planurile concrete ale vietii. |
| 20-29 | 2 | Relatie cu planurile subtile si prelucrare karmica. |
| 30-39 | 3 | Relationare, influenta asupra mediului si transmiterea intelepciunii. |
| 40-48 | 4 | Stapanirea resurselor materiale si umane; intelegerea materiei si creatiei. |

Pentru caile `4-9`, se considera un `0` implicit in fata. Omul poate intalni
putine obstacole sau ajutoare prescrise, dar poate primi contexte speciale sau o
multime de posibilitati. Directia principala este prezenta constienta.

Pentru caile `10-19`, obstacolele si ajutoarele sunt de tip `1`: atentia pe sine,
desavarsirea caracterului si asumarea propriei pozitii in lume.

Pentru caile `20-29`, obstacolele si ajutoarele sunt de tip `2`: raportarea la
planurile subtile, karma, continuitatea spiritului si relatia dintre aceasta
viata si alte straturi ale existentei.

Pentru caile `30-39`, obstacolele si ajutoarele sunt de tip `3`: relationare,
influentarea mediului, transmiterea intelepciunii si rolul de dascal asumat.

Pentru caile `40-48`, obstacolele si ajutoarele sunt de tip `4`: resurse
materiale si umane, baza creatiei, materia densa si intelegerea rostului ei.

## Observatii

Calea destinului nu se reduce complet la 1-9. Daca este nevoie de vibratia finala
a traseului, se foloseste `vibratia destinului`.

La interpretare se poate privi:

- prima cifra;
- a doua cifra;
- numarul intreg al caii destinului;
- vibratia destinului catre care conduce calea;
- raportul dintre ajutoare, obstacole si sansele primite.
