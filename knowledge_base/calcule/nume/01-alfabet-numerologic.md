# Alfabet numerologic

Pentru proiect folosim alfabetul pitagoreic.

| Numar | Litere |
| --- | --- |
| 1 | A, J, S |
| 2 | B, K, T |
| 3 | C, L, U |
| 4 | D, M, V |
| 5 | E, N, W |
| 6 | F, O, X |
| 7 | G, P, Y |
| 8 | H, Q, Z |
| 9 | I, R |

## Diacritice

Pentru numele romanesti, diacriticele se normalizeaza la litera de baza:

- A, Ă, Â -> A;
- I, Î -> I;
- S, Ș -> S;
- T, Ț -> T.

Semnele de punctuatie, spatiile si cratimele nu intra in calcul.
