# Numarul activ - exemple

## Exemplu - Elena Doinas

Prenume folosit:

```text
Elena
```

Valori:

```text
E=5, L=3, E=5, N=5, A=1
```

Calcul:

```text
5 + 3 + 5 + 5 + 1 = 19
1 + 9 = 10
1 + 0 = 1
```

Rezultat:

```text
numarul activ = 1
```

## Exemplu - Bianca Maria

Prenume complet:

```text
Bianca Maria
```

Prenume activ declarat:

```text
Bianca
```

Pentru numarul activ se calculeaza prenumele activ declarat.

Valori pentru prenumele activ:

```text
B=2, I=9, A=1, N=5, C=3, A=1
```

Calcul:

```text
2 + 9 + 1 + 5 + 3 + 1 = 21
2 + 1 = 3
```

Rezultat:

```text
numarul activ pentru Bianca = 3
```
