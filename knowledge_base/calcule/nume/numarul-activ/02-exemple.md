# Numarul activ - exemple

## Exemplu - Elena Doinas

Prenume folosit:

```text
Elena
```

Valori:

```text
E=5, L=3, E=5, N=5, A=1
```

Calcul:

```text
5 + 3 + 5 + 5 + 1 = 19
1 + 9 = 10
1 + 0 = 1
```

Rezultat:

```text
numarul activ = 1
```

## Exemplu - Bianca / Bibi

Prenume oficial:

```text
Bianca
```

Porecla folosita efectiv:

```text
Bibi
```

Pentru numarul activ se poate calcula forma folosita in viata de zi cu zi.

Valori pentru porecla:

```text
B=2, I=9, B=2, I=9
```

Calcul:

```text
2 + 9 + 2 + 9 = 22
2 + 2 = 4
```

Rezultat:

```text
numarul activ pentru Bibi = 4
```
