# Cicluri de 7 ani - exemple

Aceste exemple folosesc formula operationala curenta:

```text
ciclu 7 ani = floor(varsta / 7) + 1
pozitie in ciclu = varsta % 7 + 1
```

Formula nu este validata printr-o sursa bibliografica interna directa, dar este
folosita ca ritm secundar de praguri de varsta.

## Exemplul 1: varsta 28

Calcul:

```text
floor(28 / 7) + 1 = 4 + 1 = 5
28 % 7 + 1 = 0 + 1 = 1
```

Rezultat:

- ciclul 5;
- pozitia 1 in ciclul de 7 ani.

Interpretare scurta:

Persoana se afla la inceputul unui nou ciclu operational de 7 ani. Interpretarea
se poate folosi ca reper secundar de prag, mai ales cand se suprapune peste un
an important, un pinaclu sau o schimbare din ciclul de 9 ani.

## Exemplul 2: varsta 34

Calcul:

```text
floor(34 / 7) + 1 = 4 + 1 = 5
34 % 7 + 1 = 6 + 1 = 7
```

Rezultat:

- ciclul 5;
- pozitia 7 in ciclul de 7 ani.

Interpretare scurta:

Persoana se afla la finalul ciclului 5 operational. Aceasta pozitie poate fi
notata ca etapa de sinteza si pregatire a trecerii catre urmatorul prag de 7
ani, cu prudenta privind interpretarea simbolica.
