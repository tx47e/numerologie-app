# Tema anului - metoda

Formula adoptata pentru proiect:

```text
tema anului = interpretarea vibratiei anului
```

Tema anului nu este un calcul separat in prima versiune. Ea este stratul de
interpretare aplicat peste `vibratia anului`.

## Pasi de calcul

1. Se calculeaza vibratia anului.
2. Se cauta semnificatia rezultatului in `semnificatii.md`.
3. Se formuleaza tema practica a anului.
