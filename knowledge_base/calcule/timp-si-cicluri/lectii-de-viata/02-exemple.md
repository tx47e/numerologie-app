# Lectii de viata - exemple

## Exemplul 1: 6.11.1984

Data nasterii:

```text
zi = 6
luna = 11
an = 1984
```

Calcul:

```text
6 x 11 x 1984 = 130944
```

Sirul lectiilor:

```text
1, 3, 0, 9, 4, 4
```

Aplicare pe primii 12 ani de viata:

| An de viata | Lectie |
|---:|---:|
| 1 | 1 |
| 2 | 3 |
| 3 | 0 |
| 4 | 9 |
| 5 | 4 |
| 6 | 4 |
| 7 | 1 |
| 8 | 3 |
| 9 | 0 |
| 10 | 9 |
| 11 | 4 |
| 12 | 4 |

Interpretare scurta:

- anul 1: lectie de initiativa si identitate;
- anul 2: lectie de exprimare;
- anul 3: lectie de potential, pauza sau resetare;
- anul 4: lectie de incheiere si integrare;
- anii 5 si 6: lectii repetate de structura si disciplina.

Pentru ca lectia `4` apare in doi ani consecutivi, ea poate aduce un plus de
cariera, faima si bani. Persoana sta mai mult in aceasta lectie si are timp sa
invete mai bine ordinea, munca, structura si constructia concreta.

## Exemplul 3: calcul pentru un an de viata anume

Pentru data `6.11.1984`, sirul este:

```text
1, 3, 0, 9, 4, 4
```

Lungimea sirului este `6`.

Pentru anul 25 de viata:

```text
pozitie = ((25 - 1) mod 6) + 1
pozitie = (24 mod 6) + 1
pozitie = 0 + 1
pozitie = 1
```

Lectia anului 25 de viata este prima cifra din sir:

```text
1
```

Pentru anul 26 de viata:

```text
pozitie = ((26 - 1) mod 6) + 1
pozitie = (25 mod 6) + 1
pozitie = 1 + 1
pozitie = 2
```

Lectia anului 26 de viata este a doua cifra din sir:

```text
3
```

## Exemplul 4: relatie intre varsta si an de viata

Pentru o persoana nascuta pe `6.11.1984`:

```text
varsta implinita 0 -> anul 1 de viata -> lectia 1
varsta implinita 1 -> anul 2 de viata -> lectia 3
varsta implinita 2 -> anul 3 de viata -> lectia 0
varsta implinita 25 -> anul 26 de viata -> lectia 3
```

Regula:

```text
an_de_viata = varsta_implinita + 1
```
