# Lectii de viata

Lectiile de viata descriu o succesiune ciclica de teme anuale, calculata din
data nasterii prin inmultirea zilei, lunii si anului nasterii.

Formula adoptata in proiect este:

```text
lectii de viata = cifrele rezultatului obtinut din zi x luna x an
```

Sirul rezultat se aplica peste anii de viata, in ordine, apoi se repeta de la
inceput cand se termina.
