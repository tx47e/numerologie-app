# Vibratia anului - metoda

Formula adoptata pentru proiect:

```text
vibratia anului = reducere_numerologica(ziua nasterii + luna nasterii + anul analizat)
```

Acest calcul este echivalent cu anul personal. El se calculeaza pentru un an
calendaristic concret.

## Pasi de calcul

1. Se preiau ziua si luna nasterii.
2. Se preia anul analizat.
3. Se aduna cifrele zilei, lunii si anului analizat.
4. Se reduce rezultatul la 1-9.

Exemplu:

```text
24.04 pentru anul 2026
2 + 4 + 0 + 4 + 2 + 0 + 2 + 6 = 20
2 + 0 = 2

vibratia anului = 2
```
