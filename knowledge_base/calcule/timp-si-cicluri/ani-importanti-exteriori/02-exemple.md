# Ani importanti exteriori - exemple

Aceste exemple folosesc metoda din bibliografie:

```text
an_exterior_urmator = an_curent + suma_cifrelor(an_curent)
```

Suma cifrelor anului nu se reduce la o singura cifra.

## Exemplul 1: anul nasterii 1988

Calcul:

```text
1988: 1 + 9 + 8 + 8 = 26
1988 + 26 = 2014

2014: 2 + 0 + 1 + 4 = 7
2014 + 7 = 2021

2021: 2 + 0 + 2 + 1 = 5
2021 + 5 = 2026

2026: 2 + 0 + 2 + 6 = 10
2026 + 10 = 2036
```

Ani importanti exteriori rezultati:

- 2014;
- 2021;
- 2026;
- 2036.

Interpretare scurta:

Acesti ani pot marca perioade in care viata exterioara aduce contexte care cer
raspuns: schimbari de mediu, decizii vizibile, presiuni, oportunitati sau
adaptari concrete. Nu se citesc ca evenimente obligatorii, ci ca ani in care
interactiunea cu lumea poate deveni mai puternica.

## Exemplul 2: interval analizat 2024-2045

Pentru o persoana nascuta in `1988`, din secventa de mai sus si continuarea ei
se pastreaza doar anii care intra in intervalul `2024-2045`.

Continuare:

```text
2036: 2 + 0 + 3 + 6 = 11
2036 + 11 = 2047
```

Ani importanti exteriori in intervalul `2024-2045`:

- 2026;
- 2036.

Interpretare scurta:

In `2026`, schimbarea exterioara poate cere o reactie mai rapida, pentru ca anul
apare dupa o suma mica a anului precedent din secventa. In `2036`, tema poate fi
mai ampla sau mai structurala, deoarece pasul anterior a fost mai lung. Aceasta
diferenta nu schimba formula, dar poate ajuta la nuantarea ritmului dintre ani.

## Exemplul 3: anul nasterii 1990

Calcul:

```text
1990: 1 + 9 + 9 + 0 = 19
1990 + 19 = 2009

2009: 2 + 0 + 0 + 9 = 11
2009 + 11 = 2020

2020: 2 + 0 + 2 + 0 = 4
2020 + 4 = 2024

2024: 2 + 0 + 2 + 4 = 8
2024 + 8 = 2032

2032: 2 + 0 + 3 + 2 = 7
2032 + 7 = 2039
```

Ani importanti exteriori rezultati:

- 2009;
- 2020;
- 2024;
- 2032;
- 2039.
