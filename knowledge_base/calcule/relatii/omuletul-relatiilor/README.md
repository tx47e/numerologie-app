# Omuletul relatiilor

Omuletul relatiilor este o reprezentare sintetica a felului in care doua sau
trei persoane participa la o relatie. Diagrama porneste de la imaginea omului
vitruvian: cercul arata campul relatiei, patratul arata cadrul concret, iar
pentagrama cu varful in sus arata dinamica vie dintre persoane.
