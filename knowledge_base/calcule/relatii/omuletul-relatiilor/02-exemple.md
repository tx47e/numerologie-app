# Omuletul relatiilor - exemple

## Exemplu de calcul pentru doua persoane

Persoana A: `06.11.1984`

```text
06111984 + 30 + 3 + 18 + 9
```

Persoana B: `17.04.1984`

```text
17041984 + 34 + 7 + 32 + 5
```

Vibratiile interioare reduse:

```text
06 -> 6
17 -> 8
```

Sinteza relatiei:

```text
ce se poate realiza impreuna = 6 + 8 = 14 -> 5
ce este de rezolvat impreuna = |6 - 8| = 2
```

## Exemplu Birsan Daniel Robert si Roman Andreea Maria

Daniel:

```text
19021998 + 39 + 12 + 37 + 10
```

Andreea:

```text
12011998 + 31 + 4 + 29 + 11
```

Vibratii interioare:

```text
Daniel: 19 -> 1
Andreea: 12 -> 3
```

Sinteza relatiei:

```text
ce se poate realiza impreuna = 1 + 3 = 4
ce este de rezolvat impreuna = |1 - 3| = 2
```

Citire scurta:

- `4` ca realizare comuna arata ca relatia poate construi forma, stabilitate,
  reguli si proiecte concrete.
- `2` ca zona de rezolvat arata ca relatia cere rabdare, ascultare,
  cooperare, diplomatie si gestionarea sensibilitatii.
