# Karma

Acest director contine calculele karmice folosite in proiect.

## Calcule

- `karma-personala/`: lectii karmice si datorii karmice extrase din nume si data.
- `karma-neamului/`: teme mostenite din numele de familie.

Formulele sunt adoptate ca versiune de lucru si trebuie validate editorial inainte
de publicarea finala.
