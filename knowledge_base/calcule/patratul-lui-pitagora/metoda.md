# Patratul lui Pitagora - metoda

De stabilit metoda exacta folosita.

## Pasi generali

1. Se porneste de la data de nastere.
2. Se extrag cifrele relevante.
3. Se calculeaza numerele intermediare, daca metoda aleasa le foloseste.
4. Se completeaza matricea 3x3.
5. Se interpreteaza prezenta, absenta si repetitia cifrelor.
