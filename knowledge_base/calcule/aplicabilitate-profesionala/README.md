# Aplicabilitate profesionala

Acest director contine calculul care leaga data nasterii de Arcanele Majore,
pentru a observa doua directii:

- obstacolele si ajutoarele de pe calea implinirii destinului;
- aplicabilitatea profesionala, adica meserii, domenii si moduri potrivite de lucru.

Rezultatul nu se citeste ca eticheta fixa, ci ca arcana de lucru.
