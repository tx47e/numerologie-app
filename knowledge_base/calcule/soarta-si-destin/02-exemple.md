# Soarta si destin - exemple

## Exemplu 1: 09.02.1969

### Calcul soarta

Data se scrie cu ziua si luna pe doua cifre:

```text
09.02.1969 -> ZZLL = 0902
0902 x 1969 = 1776038
```

Rezultat:

| Camp | Valoare |
| --- | --- |
| Numar soarta | 1776038 |
| Numar de cifre | 7 |
| Completare cu zero in fata | nu este necesara |

Zona de confort:

```text
1 + 7 + 7 + 6 + 0 + 3 + 8 = 32
32 / 7 = 4,57
```

### Calcul destin

Pentru destin, zerourile se inlocuiesc cu `1`:

```text
0902 -> 1912
1969 -> 1969
1912 x 1969 = 3764728
```

Rezultat:

| Camp | Valoare |
| --- | --- |
| Numar destin | 3764728 |
| Numar de cifre | 7 |
| Completare cu zero in fata | nu este necesara |

Zona de confort:

```text
3 + 7 + 6 + 4 + 7 + 2 + 8 = 37
37 / 7 = 5,28
```

### Interpretare comparativa

Soarta are zona de confort `4,57`, iar destinul urca la `5,28`. Tema vietii poate
fi formulata ca trecere de la un cadru primit mai stabil, mai prudent si mai
asezat, catre o directie de implinire care cere mai multa miscare, asumare si
crestere. Persoana nu este chemata sa fuga de cadrul primit, ci sa-l foloseasca
drept baza pentru o varianta mai activa a propriei vieti.

## Exemplu 2: 24.07.1988

### Calcul soarta

```text
24.07.1988 -> ZZLL = 2407
2407 x 1988 = 4785116
```

Zona de confort:

```text
4 + 7 + 8 + 5 + 1 + 1 + 6 = 32
32 / 7 = 4,57
```

### Calcul destin

```text
ZZLL = 2407 -> 2417
AAAA = 1988 -> 1988
2417 x 1988 = 4804996
```

Zona de confort:

```text
4 + 8 + 0 + 4 + 9 + 9 + 6 = 40
40 / 7 = 5,71
```

## Exemplu 3: completare la 7 cifre

Pentru soarta:

```text
01.03.1985 -> ZZLL = 0103
0103 x 1985 = 204455
```

Rezultatul are 6 cifre, deci se completeaza cu zero in fata:

```text
0204455
```

Pentru destin:

```text
ZZLL = 0103 -> 1113
AAAA = 1985 -> 1985
1113 x 1985 = 2209305
```

Acest rezultat are deja 7 cifre.
