# Soarta - exemple

## Exemplu 1

Data nasterii:

```text
24.04.1982
```

Calcul:

```text
2 + 4 + 0 + 4 + 1 + 9 + 8 + 2 = 30
3 + 0 = 3
```

Rezultat:

- Soarta: 3.

Interpretare scurta:

```text
Soarta 3 indica un drum de expresie, comunicare si creativitate. Persoana este
chemata sa invete cum sa isi transforme trairile si ideile in forma vizibila.
```

## Exemplu 2

Data nasterii:

```text
17.11.1990
```

Calcul:

```text
1 + 7 + 1 + 1 + 1 + 9 + 9 + 0 = 29
2 + 9 = 11
1 + 1 = 2
```

Rezultat:

- Soarta: 2.

Observatie:

```text
Valoarea intermediara 11 poate fi notata ca informatie secundara, dar in regula
curenta a proiectului rezultatul final se reduce la 1-9.
```

## Exemplu 3

Data nasterii:

```text
05.08.1976
```

Calcul:

```text
0 + 5 + 0 + 8 + 1 + 9 + 7 + 6 = 36
3 + 6 = 9
```

Rezultat:

- Soarta: 9.

Interpretare scurta:

```text
Soarta 9 indica un drum de integrare, compasiune si finalizare. Persoana este
chemata sa invete eliberarea, perspectiva larga si folosirea experientei cu
sens.
```
