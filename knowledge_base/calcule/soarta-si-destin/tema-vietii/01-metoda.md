# Tema vietii - metoda

Formula adoptata pentru proiect:

```text
tema vietii = reducere_numerologica(soarta + destin)
```

Tema vietii combina axa datei de nastere cu axa numelui. Ea nu inlocuieste
soarta sau destinul, ci arata tensiunea sau directia comuna dintre ele.

## Pasi de calcul

1. Se calculeaza `soarta` din data nasterii.
2. Se calculeaza `destin` din numele complet.
3. Se aduna cele doua rezultate.
4. Se reduce suma la 1-9.
