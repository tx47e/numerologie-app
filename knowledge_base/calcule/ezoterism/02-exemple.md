# Ezoterism - exemple

## Exemplu 1: 30.10.1963

Data se scrie ca numar:

```text
30.10.1963 -> 30101963
```

Se face impartirea principala:

```text
30101963 / 7 = 4300280,428571...
```

Prima cifra de dupa virgula este `4`.

Rezultat principal:

| Camp | Valoare |
| --- | --- |
| Cod ezoteric principal | 4 |
| Tip principal | ezoterism practic |

Interpretare scurta:

Codul `4` indica maestrul practic: logica aplicata, experienta, intelegerea
felului in care functioneaza lucrurile si capacitatea de prognoza concreta.

Se calculeaza domeniul secundar folosind partea intreaga din prima impartire:

```text
4300280 / 7 = 614325,714285...
```

Prima cifra de dupa virgula este `7`.

Rezultat secundar:

| Camp | Valoare |
| --- | --- |
| Cod secundar | 7 |
| Domeniu in ezoterism practic | magie |

Rezultat final:

```text
cod principal 4 -> ezoterism practic
cod secundar 7 -> magie
```

## Exemplu 2: 06.11.1984

Data se scrie ca numar:

```text
06.11.1984 -> 6111984
```

Se face impartirea principala:

```text
6111984 / 7 = 873140,571428...
```

Prima cifra de dupa virgula este `5`.

Rezultat principal:

| Camp | Valoare |
| --- | --- |
| Cod ezoteric principal | 5 |
| Tip principal | ezoterism stiintific |

Interpretare scurta:

Codul `5` indica o minte orientata spre semne, simboluri, structuri, memorie si
interpretare. Persoana poate citi detalii subtile si poate lucra cu predictii
sau corelatii.

Se calculeaza domeniul secundar:

```text
873140 / 7 = 124734,285714...
```

Prima cifra de dupa virgula este `2`.

Rezultat secundar:

| Camp | Valoare |
| --- | --- |
| Cod secundar | 2 |
| Domeniu in ezoterism stiintific | terapii complementare |

Rezultat final:

```text
cod principal 5 -> ezoterism stiintific
cod secundar 2 -> terapii complementare
```

## Exemplu 3: cod principal 0

Daca impartirea datei la `7` este exacta sau prima cifra de dupa virgula este
`0`, codul principal se noteaza ca `0`.

Forma de calcul:

```text
data_nasterii_scrisa_ca_numar / 7 = numar_intreg,000...
```

Rezultat principal:

| Camp | Valoare |
| --- | --- |
| Cod ezoteric principal | 0 |
| Tip principal | cod special |

Interpretare scurta:

Codul `0` nu se citeste ca spiritual, stiintific sau practic. El indica pagina
noua, cod sters, acces neclar sau lipsa unui scenariu clar in aceasta zona.

In lucrare se noteaza prudenta si nu se forteaza concluzii despre capacitati
extrasenzoriale.
