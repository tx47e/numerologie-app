# Ezoterism

Acest director contine metoda numerologica pentru identificarea inclinatiilor
ezoterice ale unei persoane pe baza datei de nastere.

Calculul este folosit ca sectiune suplimentara in lucrarea numerologica. El
poate indica tipul de perceptie subtila, domeniile ezoterice care se pot
deschide mai usor si atentia necesara in lucrul cu informatii sensibile.

Metoda are doua etape:

1. codul ezoteric principal, obtinut din prima cifra de dupa virgula la
   impartirea datei de nastere la `7`;
2. codul secundar, obtinut din prima cifra de dupa virgula la impartirea partii
   intregi din primul calcul la `7`.
