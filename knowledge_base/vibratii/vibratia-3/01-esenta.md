# Vibratia 3 - Esenta

Vibratia 3 reprezinta principiul expresiei, al creativitatii si al comunicarii.
Este energia prin care trairea interioara capata forma: cuvant, gest, imagine,
umor, poveste, cantec sau idee impartasita.

3 aduce miscare, sociabilitate si dorinta de a face experienta umana mai vie. Ea
transforma continutul emotional in expresie si cauta sens prin joc, frumusete si
participare.
