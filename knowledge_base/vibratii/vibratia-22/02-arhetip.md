# Vibratia 22 - Arhetip

Arhetipul vibratiei 22 este Marele Constructor.

Acest arhetip gandeste in structuri mari: proiecte, comunitati, institutii,
fundatii, metode si sisteme care pot ramane in urma lui. El simte ca ideea nu
este suficienta daca nu devine forma.

Matur, 22 poate construi cu rabdare si viziune. Imatur, se poate simti coplesit
de dimensiunea propriei mize si poate alterna intre ambitie uriasa si blocaj.
