# Vibratia 7 - Arhetip

Arhetipul vibratiei 7 este Cautatorul. Cautatorul investigheaza, contempla si cauta
adevarul, fie prin stiinta, fie prin filosofie, spiritualitate sau observatie
interioara.

In forma matura, Cautatorul devine intelept si discernamant. In forma imatura,
poate deveni izolat, sceptic excesiv sau rupt de viata concreta.
