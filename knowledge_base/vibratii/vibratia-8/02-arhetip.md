# Vibratia 8 - Arhetip

Arhetipul vibratiei 8 este Strategul. Strategul intelege sisteme, resurse si
consecinte. El stie ca intentia buna nu este suficienta daca nu exista si executie,
responsabilitate si masura.

In forma matura, Strategul conduce cu integritate. In forma imatura, poate cauta
control, statut sau succes fara raportare la valori.
