# Vibratia 8 - Arhetip

Arhetipul vibratiei 8 este Strategul. Strategul intelege sisteme, resurse si
consecinte. El stie ca intentia buna nu este suficienta daca nu exista si executie,
responsabilitate si masura.

In forma matura, Strategul conduce cu integritate. In forma imatura, poate cauta
control, statut sau succes fara raportare la valori.

Strategul se recunoaste in felul in care citeste resursele, mizele si raporturile
de forta. El vede unde se pierde energie, unde este nevoie de decizie si ce fel de
structura poate transforma potentialul in rezultat.

Lectia lui este sa foloseasca puterea fara sa se confunde cu ea. Autoritatea
matura nu are nevoie sa intimideze. Cand este integrat, Strategul construieste
rezultate vizibile, dar pastreaza masura, responsabilitatea si respectul pentru
oameni.
