# Vibratia 11 - Directii de dezvoltare

- Practica ancorarea: rutina, corp, respiratie, ordine emotionala.
- Noteaza intuitiile si verifica-le in timp.
- Evita deciziile luate exclusiv din anxietate sau exaltare.
- Dezvolta un limbaj clar pentru ceea ce simti.
- Foloseste creatia, consilierea, educatia sau comunicarea ca forme de expresie.
- Pastreaza legatura cu vibratia 2: cooperare, tact, relatie si ascultare.

Practici utile:

- notarea intuitiilor si verificarea lor in timp;
- respiratie, corp sau rutina pentru reglare emotionala;
- exprimarea mesajelor subtile in limbaj simplu;
- alegerea contextelor in care sensibilitatea este respectata, dar si maturizata.
